# 📊 Sales Performance Dashboard (Excel Project)

## 🔍 Overview
This project analyzes sales performance of executives across different regions using Excel.

## 📁 Dataset
The dataset includes:
- Daily Sales (Day 1 to Day 5)
- Total Sales
- Target
- Target Hit %
- Away From Target %

## 📊 Key Insights
- Identified top-performing sales executives
- Compared target achievement across regions
- Highlighted performance gaps

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboard
